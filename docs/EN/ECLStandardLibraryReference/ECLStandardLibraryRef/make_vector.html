make_vector

STD.BLAS.make_vector ( m, v );

mNumber of elements
vThe values, default is 1
Return:The vector

The make_vector function generates a vector of dimension n

Example:

IMPORT STD;
twos_vector := STD.BLAS.make_vector(4, 2); // a vector of [2, 2, 2, 2]
