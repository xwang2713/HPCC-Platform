DaliServer

STD.System.Thorlib.DaliServer ( )

Return: Daliserver returns a VARSTRING value.

The Daliserver function returns the IP and port of the system data store (Dali) server for the environment running the workunit.

Example:

A := Thorlib.Daliserver();
