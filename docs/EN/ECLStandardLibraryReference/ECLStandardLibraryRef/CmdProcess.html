CmdProcess

result := STD.System.Util.CmdProcess ( program, input );

programA null-terminated string containing the name of the program to execute. This may include command-line parameters.
inputA string containing the text to pipe into the program through stdin.
Return:CmdProcess returns returns a STRING value.

The CmdProcess function pipes the input text to the specified program. This is similar to the PIPE built-in function, but limited to simple text input and output.

Example:

IMPORT STD;
   
OUTPUT(STD.System.Util.CmdProcess('cat','George Jetson'));
